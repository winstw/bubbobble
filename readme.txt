     =============
     BUBBLE BOBBLE
     =============
======================================
TP de programmation par Geoffroy LARUE
======================================


Pour compiler le jeu, rendez-vous dans le dossier "game/src" en console et exécutez
la commande "make".

Retournez dans le dossier parent "game".

Si tout s'est bien déroulé, un binaire bubbob a été créé dans ce répertoire.

Pour lancer le jeu, il suffit de le lancer avec "./bubbob" depuis ce répertoire.

=======================================================================================
To compile the Bubble Bobble game, you simply need to get into the "game/src" folder
into a console, and run "make".
Go back to the main "game" folder where you should find a "bubbob" binary file if
the compilation succeeded.

To launch the game, get in the main "game" folder and run it with "./bubbob".

=======================================================================================
